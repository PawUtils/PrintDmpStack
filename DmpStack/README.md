# DumbPrograms.DmpStack

JSON serialization related classes to consume the output from
[PrintDmpStack.exe](https://github.com/PawUtils/PrintDmpStack),
a Windows program to print the stack from a crash dump (.dmp) file.

1. `DmpStack.Frame`
1. `DmpStack.FramesJsonSerializationContext`

## Build

Open the `PrintDmpStack.slnx` file in Visual Studio.
